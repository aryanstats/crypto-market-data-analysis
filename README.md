# Crypto Market Data Analysis

## Overview
This project analyzes real-time cryptocurrency market data using the CoinMarketCap API.
The goal is to explore market capitalization, trading volume, and short-term price volatility.

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- CoinMarketCap API
- Jupyter Notebook

## Key Analysis
- Top cryptocurrencies by market capitalization
- Price vs trading volume relationship
- 24-hour price volatility analysis
- Market dominance and concentration insights

## Key Insights
- Bitcoin dominates the crypto market by market capitalization
- Stablecoins show high trading volume despite low price volatility
- Smaller cryptocurrencies exhibit higher short-term volatility

## Data Source
CoinMarketCap API (free tier)
